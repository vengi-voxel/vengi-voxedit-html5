This is the alpha version of vengi voxedit for the web. Expect bugs to happen - you can load and save files to your local filesystem.
Download Mill.vengi and load it afterwards in voxedit via File -> Open (Animated model of a windmill - voxelized from quaternius.com)
There are native versions of the voxedit available for Windows, OSX and Linux.
If you find issues, please report them and help me to fix them.
Find more information about vengi and the other voxel related tools here.
